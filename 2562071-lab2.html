Latest Project

The last project I worked on was a website for my dad’s construction company. The goal was to create a clean, modern portfolio that showcases his company's past and ongoing projects. I built a dynamic project gallery where visitors can explore detailed information about each project, view high-quality images, and easily navigate through the gallery with interactive thumbnails and navigation arrows. The website is fully responsive, making it easy to browse on both desktop and mobile devices. I focused on creating a user-friendly experience with clear project categories and status updates, helping potential clients and partners get a comprehensive view of my dad’s work. This project was a great opportunity for me to apply my web development skills and create something that truly represents my dad’s business.
